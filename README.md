## Usage

```
git clone git@github.com:kami-blue/bot.git kamiBlueBot
cd kamiBlueBot
npm i
node index.js
```

Requirements: 
```
node >=12
git(latest)
go(latest)
GNU make(latest)

//on Linux
Run the install.sh file.

//on Windows
run the install.bat file.

//on MacOS
Run the install.tool file.
```

`auth.json` example:
```json
{
  "token": "tokenhere",
  "githubtoken": "tokenhere"
}
```
